#### 프로젝트 소개
중간 프로젝트 주제였던 북적북적(도서 쇼핑몰)을 1차 최종 프로젝트로 보완합니다.

>구현

- 회원가입
- 로그인(소셜API사용 포함)
- 검색
- 장바구니
- 주문
- 결제
- 관리자페이지
